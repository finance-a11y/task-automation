## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| bot → ClickUp REST | outbound reads of field/member config; response is untrusted JSON |
| bot → Redis | cached config crosses the process boundary and is later trusted by the resolver |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-06-01 | Tampering | ClickUp field/member JSON | mitigate | Shape-guard every nested access; reject malformed payloads with a typed error instead of trusting undefined into the cache |
| T-06-02 | Info Disclosure | error bodies / logs | mitigate | Thrown errors carry status + body text only — NEVER the Authorization token (asserted in tests), matching createTask/getTask |
| T-06-03 | DoS | Retry-After abuse | accept | Already bounded by the existing createRetryingFetch clamp (MAX_RETRY_AFTER_MS=30s); reused unchanged |
