|----------|-------------|
| OpenAI API → parseTask | Model output is untrusted text; structured outputs + schema validation is the gate |
| OPENAI_API_KEY → client | Secret crosses into the SDK; must never be logged or read at module load |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-05 | Tampering | parseTask model output | mitigate | json_schema strict / zodResponseFormat guarantees shape; ParsedTaskSchema.parse re-validates; malformed → typed ParseError, never an unvalidated object (Pitfall 4) |
| T-02-06 | Information disclosure | OPENAI_API_KEY | mitigate | createOpenAIClient is lazy (mirrors createRedis); key read only from injected env, never logged; errors log message text only |
| T-02-07 | Elevation/Spoofing | LLM emitting IDs | mitigate | Prompt + schema constrain the model to raw human strings only; ID resolution is the deterministic resolver's job (LOCKED CONTEXT) |
</threat_model>
