      1. `const raw = await req.text()` FIRST — never JSON-parse before computing the HMAC.
      2. `const sig = getClickUpSignatureHeader(req.headers)` (case-insensitive, NOTIFY-01).
      3. If `!sig || !verifyClickUpSignature(raw, sig, env.CLICKUP_WEBHOOK_SECRET)` → return
         `new Response("invalid signature", { status: 401 })` (reject forged/missing).

      5. ACK fast: build deps and call `waitUntil(processClickUpWebhook(deps, payload))`, then return
         `new Response("ok", { status: 200 })`. Do NOT await the processing before responding (CONTEXT >
