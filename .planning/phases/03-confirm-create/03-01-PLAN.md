ClickUp REST v2 facts verified during planning:
- Base URL: https://api.clickup.com/api/v2
- Create task: POST /list/{listId}/task
- Auth header is the raw token: `Authorization: <CLICKUP_API_TOKEN>` (personal token — NOT "Bearer ")
- start_date / due_date are epoch MILLISECONDS (integers). Pair each with start_date_time/due_date_time=false (resolver emits day-granularity midnight-in-TZ ms).
- assignees is an array of numeric member ids.
- custom_fields on create is an array of {id, value}. Dropdown value = option UUID string; URL field value = the link string. Confirmed: setting custom fields inline on create is supported, so NO separate /task/{id}/field/{fieldId} call is needed.
- Link/Loom url custom field id = 5a03e7cb-0af0-4179-9f05-d0620334fc08
- Success response JSON contains `id` (string) and `url` (string).
